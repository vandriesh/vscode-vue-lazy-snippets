# vscode-vue-lazy-snippets README

This is for lazy developers who might not yet use tests in their development.

## Features

Snippets :

| Snippet                                      | Purpose                                                      |
| -------------------------------------------- | ------------------------------------------------------------ |
|`tvcwp`                                       | test a vue component with a property                         |


## Requirements

Snippets target env is `vue`, `jest` and `typescript`.
